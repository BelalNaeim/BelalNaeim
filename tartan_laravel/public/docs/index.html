Introduction

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Base URL

https://tartan.aquadic.com

Authenticating requests

This API is not authenticated.

Auth

SignUp User and return JWT Token

Example request:
curl --request POST \
    "https://tartan.aquadic.com/api/auth/signup" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"John Doe\",
    \"email\": \"medical@aquadic.com\",
    \"phone\": \"+201101782890\",
    \"password\": \"JamesBond007\"
}"

const url = new URL(
    "https://tartan.aquadic.com/api/auth/signup"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "John Doe",
    "email": "medical@aquadic.com",
    "phone": "+201101782890",
    "password": "JamesBond007"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$response = $client->post(
    'https://tartan.aquadic.com/api/auth/signup',
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'json' => [
            'name' => 'John Doe',
            'email' => 'medical@aquadic.com',
            'phone' => '+201101782890',
            'password' => 'JamesBond007',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));

GET api/playgrounds

Example request:
curl --request GET \
    --get "https://tartan.aquadic.com/api/playgrounds?keyword=%22Somuha%22.&lat=%2231.5656%22.&lng=%22-29.5656%22.&city_id=%221%22.&area_id=%222%22.&time=%2212%3A00%3A00%22." \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "https://tartan.aquadic.com/api/playgrounds"
);

const params = {
    "keyword": ""Somuha".",
    "lat": ""31.5656".",
    "lng": ""-29.5656".",
    "city_id": ""1".",
    "area_id": ""2".",
    "time": ""12:00:00".",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
